## House Prices Project
This project compares Regression, Classification, and Clustering techniques on the Kaggle House Prices dataset.

**Workflow Strategy:**
To organize the code Branches were used:
* "step1": Code for predicting exact house prices.
* "step2": Code for classifying houses into Cheap, Medium, and Expensive tiers.
* "step3": Code for unsupervised grouping of houses.
Once a step is completed and tested, its branch will be merged into the MAIN branch.
